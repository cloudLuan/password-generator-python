# password-generator-python

# Gerador de Senhas em Python

Projeto desenvolvido com o objetivo de praticar lógica de programação e fundamentos da linguagem Python.

O programa gera senhas aleatórias de acordo com a quantidade de caracteres informada pelo usuário.

## Funcionalidades
- Geração de senha aleatória
- Definição do tamanho da senha
- Utilização de letras maiúsculas e minúsculas
- Utilização de números

## Conceitos aplicados
- Variáveis
- Entrada e saída de dados (`input` e `print`)
- Laço de repetição (`while`)
- Biblioteca `random`
- Manipulação de strings

## Como executar
1. Tenha o Python instalado
2. Execute o arquivo `main.py`
3. Informe a quantidade de caracteres da senha

## Status do projeto
Projeto iniciante em desenvolvimento para fins de aprendizado.
